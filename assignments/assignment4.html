GitHub and VSCode Guide

Introduction

Welcome, developers! Today's assignment is designed to familiarize you with essential tasks such as setting up GitHub with VSCode and understanding the necessary steps. This foundational knowledge is crucial for your journey in software development. Let's dive into the following topics:

  1. Creating a Gmail account
  2. Creating a GitHub account
  3. Connecting VS Code to your GitHub account
  4. Cloning/pulling a repository from GitHub
  5. Pushing a repository/folder to GitHub
  6. Publishing the repository as a live site

Creating a Gmail Account

  1. Navigate to the Gmail website using your preferred web browser
  2. Initiate the account creation process by selecting the appropriate option
  3. Specify whether the account is for personal or business use
  4. Input your personal information and desired login credentials
  5. Confirm your identity through phone number verification
  6. Finalize the account setup by providing any additional required details

Creating a GitHub account

  1. Visit the GitHub website
  2. Click on the "Sign up" button to start the registration process
  3. Enter your email address, create a password, and choose a username
  4. Complete the verification process to prove you're not a robot
  5. Choose your account type (free or paid)
  6. Customize your experience by answering a few questions (optional)

Connecting VS Code to your GitHub account

  1. Install Git on your computer if not already installed
  2. Open VS Code and navigate to the Source Control view
  3. Click on the "Sign in to GitHub" button
  4. Follow the prompts to authorize VS Code in your browser
  5. Return to VS Code and verify the connection
  6. Configure your Git username and email in VS Code settings

Cloning/pulling a repository from GitHub

  1. Open VS Code and access the Command Palette
  2. Type "Git: Clone" and select it from the list
  3. Enter the URL of the GitHub repository you want to clone
  4. Choose a local directory to store the cloned repository
  5. Wait for the cloning process to complete
  6. Open the cloned repository in VS Code

Pushing a repository/folder to GitHub

  1. Open your project folder in VS Code
  2. Initialize a Git repository if not already done
  3. Stage your changes using the Source Control view
  4. Commit your changes with a descriptive message
  5. Create a new repository on GitHub if needed
  6. Push your local repository to GitHub using the VS Code interface

Publishing the repository as a live site

  1. Ensure your repository contains an index.html file
  2. Go to your repository's settings on GitHub
  3. Navigate to the "Pages" section
  4. Choose the branch you want to publish (usually main or master)
  5. Select the root folder as the publishing source
  6. Save the settings and wait for GitHub to generate your site's URL
