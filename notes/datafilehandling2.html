File Handling In C.

Question 2:

Write a C program to create a new data file and store an integer in it using putw(). Read the integer afterwards using getw()..

Code


#include <stdio.h>

int main() {
    FILE *file;
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);
                            
    file = fopen("data.bin", "wb");
    putw(num, file);
    fclose(file);
                            
    file = fopen("data.bin", "rb");
    num = getw(file);
    printf("Integer read from file: %d\n", num);
    fclose(file);
    printf("Run by Bibek Gautam\n");
    return 0;
}

Output

Enter a number: 4
Integer read from file: 4
Run by Bibek Gautam
