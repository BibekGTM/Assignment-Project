File Handling In C.

Question 1:

Write a C program to create a new data file and store a character in it using putc(). Read the character afterwards using getc().

Code


    #include <stdio.h>
    
    int main() {
        FILE *file;
        char ch = 'A';
        
        file = fopen("data.txt", "w");
        putc(ch, file);
        fclose(file);
        
        file = fopen("data.txt", "r");
        ch = getc(file);
        printf("Read character: %c\n", ch);
        fclose(file);
        
        return 0;
    }

Output

Read character: A
